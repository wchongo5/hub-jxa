# hub-jxa-projects

Shared JXA (JavaScript for Automation) utilities and project templates.

## Structure

- `libraries/` – Reusable JXA libraries
- `projects/` – Concrete automation projects using the libraries

## Requirements

- macOS
- Script Editor or `osascript`
- JavaScript for Automation (JXA)

## Usage

Each project folder is self-contained. Libraries are designed to be imported
or copied into projects as needed.
